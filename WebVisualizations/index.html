Summary: Latitude vs. X

The objective of this project was to study the relationship between climate variables of a set of cities against its location in the world. So, the main idea was to analyze the data and conclude if the Max Temperature, Cloudiness, Humidity and Wind Speed are related to the Latititude of the set of cities; the climate data was obtained from World Weather API.

In order to accomplish the objective, I upload and managed the data with Python, by using Jupyter Notebook. After cleaning the data, I used the Matplotlib library to plot the data from the climate variables versus the Latitude of the City. So, I plotted on the Y-axis the value each climate variable (Max Temperature, Cloudiness, Humidity and Wind Speed) and on the X-Axis the corresponding latitude of the City. For each plot, a description was made regarding the correltion of both variables and any observable trend.
